Exporter++ is a Burp Suite extension that is designed to provide users with the convenience of uploading identified security issues directly to Pwndoc or exporting them in a variety of document formats.

Features

  • The extension offers various export options, including formats like Pwndoc, CSV, JSON, XML, Markdown and Docx.
  • You can choose to include the full HTTP Requests/Responses of the Security issues in your exported file.
  • There is support for custom templates in the formats of Markdown (with Apache FreeMarker) or Docx (with XDocReport and Apache Velocity engine).
  • The feature of AI-enabled executive summary generation is enabled using Burp AI.
  • Exporting is not compulsory for all security issues; only the ones selected by you will be exported.
  • The exported Security issues are arranged in an ordered form sorted by severity, confidence, and name.

Usage

  1. Begin by selecting the issues you aim to export in the Target Tab.
  2. To submit your selections, right click and navigate the dropdown menu in the following order: Extensions → Exporter++ → Send to Exporter++.
  3. Proceed to the Exporter++ tab and select the export options that best suits your needs. The options available are:
    • Pwndoc
    • CSV
    • JSON
    • XML
    • Markdown, along with options of a user-supplied or an AI-generated Executive Summary
    • Docx, along with options of a user-supplied or an AI-generated Executive Summary.
  4. Click the Export button to initiate the export process.
